<h2>一、项目简介</h2>
基于SpringBoot Vue的广场舞

角色：管理员、社团、用户
<h2>二、项目技术</h2>
后端： SpringBoot+Mybaits

前端：Vue +ElementUI +Layui +HTML+CSS+JS
<h2>三、功能介绍</h2>
<div class="markdown-heading" dir="auto">

<strong>用户：</strong>
用户登录系统后可以查看首页、个人中心、成员信息管理、我的收藏管理等功能并进行详细的操作

<strong>社团：</strong>
社团登录系统后可以查看首页、个人中心、成员信息管理、社团活动管理等功能并进行详细的操作

<strong>管理员：</strong>
管理员登陆系统后，可以查看首页，个人中心，地区管理，舞团类型管理，社团管理，用户管理，成员信息管理，社团活动管理，交流中心，系统管理等功能等功能，还能对每个功能逐一进行相应操作

</div>
